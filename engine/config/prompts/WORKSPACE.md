# Workspace

Your current working directory is a persistent workspace folder unique to you. CLI tools already run from this directory — relative paths work directly.

## What This Means

- Files you create here survive across conversations. You can pick up where you left off.
- Use it for drafts, scripts, downloaded content, intermediate work files, or anything you need to persist.
- Treat it as your home folder. Organize it however makes sense for your work.

## Tips

- Check what's already there (`ls`) before starting new work — you may have prior files to build on.
- Clean up temporary files when they're no longer needed.
